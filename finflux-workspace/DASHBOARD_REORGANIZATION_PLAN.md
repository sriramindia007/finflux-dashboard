# DASHBOARD REORGANIZATION PLAN

## BRANCH DASHBOARD - SECTION STRUCTURE

### **SECTION 1: Branch Network Overview**
- Total Centres
- Total Groups  
- Field Staff
- Active Clients

### **SECTION 2: Operational Performance**
- Applications Received/Approved
- Disbursed Loans (Count & Value)
- Meeting Attendance %
- Geo-Fence Compliance

### **SECTION 3: Financial Metrics**
- Collection Efficiency
- Collections Paid (MTD)
- Overdue Amount
- Write-offs (YTD)

### **SECTION 4: Portfolio Quality**
- PAR 30+ / 60+ / 90+ / 180+
- Digital Collection %
- Risk Category

---

## CENTRE DASHBOARD - SECTION STRUCTURE

### **SECTION 1: Centre Network Overview**
- Total Centres
- Total Groups
- Field Officers
- Active Clients

### **SECTION 2: Workforce Metrics**
- Field Officer Count
- Attrition Rate
- Collection Compliance
- Productivity per FO

### **SECTION 3: Collection Performance**
- On-Time Collection %
- Collection Amount
- Digital vs Cash Split
- Collection Efficiency

### **SECTION 4: Portfolio Risk**
- PAR 30+ / 60+ / 90+ / 180+
- Overdue Amount
- Risk Classification

## IMPLEMENTATION APPROACH:

1. **Branch Dashboard**:
   - Keep existing KPIs but reorganize into 4 clear sections
   - Add section headings with visual separators
   - Add dividers between sections
   - Rename sections appropriately

2. **Centre Dashboard**:
   - Reorganize existing KPIs into logical groups
   - Add section headings
   - Add dividers
   - Focus on centre & workforce specific metrics

Both dashboards will follow the Home Dashboard pattern with:
- Section headings with bullet points
- Horizontal dividers (border-t border-secondary-200)
- Logical grouping
- Clear visual hierarchy
